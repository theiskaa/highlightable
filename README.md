## Usage

### _Very basic usage:_
```dart
HighlightText(                                          
  'Only numbers: [10, 25, 50, ...] will be highlighted',
  // would highlight only numbers.                      
  highlight: Highlight(pattern: r'\d'),                 
)
```

### _Custom usage:_
```dart     
HighlightText(                                   
  "Hello, Flutter!",                             
  // Would highlight only "Flutter"              
  // full word 'cause [detectWords] is enabled.  
  highlight: Highlight(                          
    words: ["Flutter"],                          
  ),                                             
  caseSensitive: true, // Turn on case-sensitive.
  detectWords: true,                             
  style: TextStyle(                              
    fontSize: 25,                                
    color: Colors.black,                         
    fontWeight: FontWeight.bold,                 
  ),                                             
  highlightStyle: TextStyle(                     
    fontSize: 25,                                
    letterSpacing: 2.5,                          
    color: Colors.white,                         
    backgroundColor: Colors.blue,                
    fontWeight: FontWeight.bold,                 
  ),                                             
)
```
